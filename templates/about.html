About

Hi, I'm RaTilicus this is my website for the 7 Days To Die Server I am hosting.
The server is hosted on the same url as this 7d2d.ratilicus.com on port 25000.

Server/Site Notes:


We are currently using the Live Free or Die mod.
While stock client works fine for the most part, the additional content, like icons, do not show properly.
Also in some locations on the map you can get stuck with the console constantly scrolling an error message, and cannot be closed.
The solution is to install the mod. Optionally, if you are logged in to this site, you can teleport yourself via this site, to a known safe location.
The Mod can be downloaded from here: https://7daystodie.com/forums/showthread.php?23773-Live-Free-or-Die.
The files need to be extracted into your 7 Days To Die directory, overriding some files. (I recommend backing up before install)

Note: The server is currently running with Anti-Cheat off. Since Alpha 11 release, my linux 32bit client of 7d2d doesn't start.
The 64bit client does work (currently), but it does not seem to support the Anti-Cheat, so I have to have it disabled, for now.

The code that runs this site and map can be found on my github page: https://github.com/ratilicus/sdtd.

Logins and passwords are offered to all players who enter the server in the welcome message.
There are a couple benefits to logging in such as:
  • Logged in player can create personalized markers (Marker with their user name) that other users cannot remove
  • Logged in player can create hidden markers (start the title of the marker with a '*' to make it hidden).
  • Logged in player can teleport to a chosen location at specified height.
  • There are other planned features that will only be available to logged in players only.

If you have any comments or suggestions feel free to contact me at rat@ratilicus.com.

Map Legend

  • green triangles = players
  • white triangles = animals: deer, pigs, rabbits
  • dark gray triangles = dead/killed animals, players, zombies
  • orange triangles = zombies
  • red triangles = dangerous zombies: cop zombies, dogs
  • yellow squares = player made markers
    - click on marker to view details
    - click anywhere on map to create a marker
    - to create a private marker only you can see put a * in front of the description (note: you have to be logged in)
  • To log in type your in-game player name, and the 4 digit number in the welcome message in the game is the password.
